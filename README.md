# 🏥 Campus Health and Wellness Companion

## 📌 Overview

The **Campus Health and Wellness Companion** is a full-stack web application designed to help university students efficiently manage their health and wellness.

It provides a centralized platform for:

* Booking medical appointments
* Tracking medications
* Managing medical records
* Monitoring daily wellness metrics

This system bridges the gap between students and campus healthcare providers by digitizing clinical workflows, reducing paperwork, and enabling proactive health management.

---

## 🚀 Key Features

### 🔐 Authentication & Security

* Secure login and registration system
* Password protection using SHA-256 hashing

### 📊 Student Dashboard & Wellness Tracker

* Visual tracking of daily health metrics
* Includes sleep, water intake, and screen time

### 📅 Appointment Scheduling

* Book consultations with campus doctors
* Select department and preferred timeslot

### 🗂 Medical Records Management

* Maintain complete health history
* Includes allergies, chronic conditions, surgeries, and reports

### 💊 Medication Management

* Track prescriptions, dosage, and duration
* Helps ensure consistency in treatment

### 🏥 Admin Portal

* Allows healthcare providers to manage and monitor student records
* Centralized access to patient data
